代码生成器

  1. 配置php命令

    1.Windows

    假如php.exe在E:\xampp\php里面。
    1.高级系统设置-系统属性-高级-环境变量
    2.Path点击编辑
    3.Path里面的变量值添加E:\xampp\php

    2.Linux

    假如php放在usr/local/php/bin/php里面。
    我们直接用命令：/usr/local/php/bin/php test.php

    3.测试Windows和Linux

    我们可以试试能不能执行php命令。
    1.我们先创建test.php
    输入以下代码: 
                                                    <?php
                                                echo 'hello world';

    windows执行命令: php E:\test.php
    linux执行命令: /usr/local/php/bin/php test.php
    如来有输出hello world说明我们配置成功了

  2. 生成控制器

    index.php --task=Soter_Generator --type=controller --name=MyFirstController
    我们可以在application/classes/Controller里面看到，有一个MyFirstController.php文件。
    打开MyFirstController.php文件我们会看到以下代码 
                                                    <?php
                                                class  Controller_MyFirstController extends Soter_Controller {
                                                        public function do_index() {

                                                        }
                                                }
    注意：index.php是我们存放soter程序的入口文件,name是文件名
    提示：
    了解详细的控制器，点击进入控制器

  6. 生成Dao

    1.首先我们先创建test数据表，代码如下 
                                           DROP TABLE IF EXISTS `test`;
                                       CREATE TABLE `test` (
                                                `id` int(11) NOT NULL AUTO_INCREMENT COMMENT '标识',
                                                `username` varchar(20) NOT NULL COMMENT '用户名',
                                                `addtime` int(10) NOT NULL COMMENT '添加时间',
                                                 PRIMARY KEY (`id`)
                                       ) ENGINE=InnoDB AUTO_INCREMENT=5 DEFAULT CHARSET=utf8;
    2.输入以下命令：
    index.php --task=Soter_Generator_Mysql --type=dao --name=MyFirstDao --table=test
    我们可以在application/classes/Dao里面看到，有一个MyFirstDao.php文件。
    打开MyFirstDao.php文件我们会看到以下代码 
                                                    <?php
                                                
                                                class Dao_MyFirstDao extends Soter_Dao {

                                                        public function getColumns() {
                                                                return array(
                                                                                'id'//标识
                                                                                ,'username'//用户名
                                                                                ,'addtime'//添加时间
                                                                                );
                                                        }

                                                        public function getPrimaryKey() {
                                                                return 'id';
                                                        }

                                                        public function getTable() {
                                                                return 'test';
                                                        }

                                                }
    注意：index.php是我们存放soter程序的入口文件,name是文件名,table是我们表名
    提示：
    了解详细的Dao，点击进入Dao

  7. 生成Bean

    1.首先我们先创建test数据表，代码如下 
                                           DROP TABLE IF EXISTS `test`;
                                       CREATE TABLE `test` (
                                                `id` int(11) NOT NULL AUTO_INCREMENT COMMENT '标识',
                                                `username` varchar(20) NOT NULL COMMENT '用户名',
                                                `addtime` int(10) NOT NULL COMMENT '添加时间',
                                                 PRIMARY KEY (`id`)
                                       ) ENGINE=InnoDB AUTO_INCREMENT=5 DEFAULT CHARSET=utf8;
    2.输入以下命令：
    index.php --task=Soter_Generator_Mysql --type=bean --name=MyFirstBean --table=test
    我们可以在application/classes/Bean里面看到，有一个MyFirstBean.php文件。
    打开MyFirstBean.php文件我们会看到以下代码 
                                                    <?php
                                                
                                                class Bean_MyFirstBean extends Soter_Bean {

                                                        //标识
                                                        private $id;

                                                        //用户名
                                                        private $username;

                                                        //添加时间
                                                        private $addtime;

                                                        public function getId() {
                                                                return $this->id;
                                                        }

                                                        public function setId($id) {
                                                                $this->id = $id;
                                                                return $this;
                                                        }

                                                        public function getUsername() {
                                                                return $this->username;
                                                        }

                                                        public function setUsername($username) {
                                                                $this->username = $username;
                                                                return $this;
                                                        }

                                                        public function getAddtime() {
                                                                return $this->addtime;
                                                        }

                                                        public function setAddtime($addtime) {
                                                                $this->addtime = $addtime;
                                                                return $this;
                                                        }

                                                }
    注意：index.php是我们存放soter程序的入口文件,name是文件名,table是我们表名
    提示：
    了解详细的Bean，点击进入Bean
