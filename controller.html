控制器

  1. 访问控制器

    控制器访问支持PathInfo模式。下面是个例子：
    这里假定控制器文件配置方法url后缀是.php，方法前缀是do，方法名首字母大写不包含前缀。
    PathInfo模式：http://127.0.0.1/index.php/Welcome/index.php?id=123
    说明：
    1.这个地址是访问控制器目录下面的classes\Controller\Welcome.php里面的Welcome类的do_index方法。
    2.同时传递了一个get变量id，可以通过Sr::get('id');获取123。

  2. 代码规范

    控制器类必须继承Soter_Controller类，类名必须带前缀Controller_，类名首字母大写。
    示例如下： 
                                                    class Controller_Welcome extends Soter_Controller {

                                                        public function __construct(){
                                                                echo '构造函数';
                                                        }

                                                        public function do_index() {
                                                                echo Sr::get('id');
                                                        }
                                                }

  3. 和控制器相关的配置


    配置在application\bootstrap.php文件里面
    1.默认控制器
    ->setDefaultController('Welcome')
    2.默认方法
    ->setDefaultController('index')
    3.方法前缀
    ->setMethodPrefix('do_')
    4.方法url后缀,比如_php或者.controller.php
    ->setMethodUriSubfix('.php')
