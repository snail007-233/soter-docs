Dao层

  1. 什么是Dao

    Dao是一个思想，如果不正确使用就跟Model一样。Dao我们可以想象成一张表，多张表就多个Dao，也可以想成绑定表。什么意思呢？
    比如：现在有张user表，那么我们对应Dao文件就是UserDao.php，那么我们UserDao.php就可以对user表增删改查。
    切勿如下操作：
    我们有user表和user_vip表，然后只有创建一个Dao文件。
    那就没有意义了，那我们还不如建Model文件，我们要遵守一个Dao文件对应一张表才可以灵活使用与扩展。

  2. Dao层规范

    1.Dao类文件都放在application/classes/Dao文件夹里面
    2.Dao必须要继承Soter_Dao抽象类，实现父类抽象方法。
    示例如下：
    2.1.新建Dao文件application/classes/Dao/TestDao.php
    2.2.输入以下代码： 
                                                    <?php

                                                class Dao_TestDao extends Soter_Dao {
                                                        
                                                        public function getTable(){}

                                                        public function getPrimaryKey(){}

                                                        public function getColumns(){}
                                                }
    可以看到Dao_TestDao需要继承3个方法，getTable、getPrimaryKey和getColumns分别什么意思呢？
    第一个getTable是获取表，比如：我们有个user表，那么函数里面可以有return 'user';。
    第二个getPrimaryKey是获取主键，比如：我们user表定义一个id字段为主键，那么函数里面可以有return 'id';。
    第三个getColumns是获取字段名称数组，比如：我们有user表里面有id、uid和username等等……字段
    示例如下: 
                                                    function getColumns(){
                                                        return array(
                                                                'id',
                                                                'uid',
                                                                'username',
                                                                ……
                                                        );
                                                }
    提示：
    以上三个方法都可以自己扩展，但是我们要记得一个Dao文件对应一个表。

    那么我们可以通过setDb()来设置另外一个数据库，现在就可以用setDb()来找到数据表了。
    $db 必填：Soter_Database_ActiveRecord对象

    2.添加数据

    通过insert($data)方法，我们可以添加数据。
    $data 必填：需要添加的数据 例如：array('username' => '张三', 'addtime' => time());
    假如我们子类要调用,如下： 
                                                    $data = array('username' => '张三', 'addtime' => time());
                                                echo parent::insert($data);
                                                //上面将输出:INSERT INTO `test` (`username`,`addtime`) VALUES (?,?)
                                                
                                                //如果我们想要直接执行sql,如下:
                                                $data = array('username' => '张三', 'addtime' => time());
                                                echo parent::insert($data)->execute();
                                                //上面将输出:1

    3.更新数据

    通过update($data, $where)方法，我们可以更新数据。
    $data 必填：需要更新的数据 例如：array('username' => '李四', 'addtime' => time());
    $where 必填：可以是$where条件关联数组，还可以是主键值。
    假如我们子类要调用,如下： 
                                                    $data = array('username' => '王五', 'addtime' => time());
                                                echo parent::update($data, 1);
                                                //上面将输出:UPDATE `test` SET `username` = ? , `addtime` = ? WHERE `id` = ?

                                                $data = array('addtime' => time());
                                                $where = array($this->getPrimaryKey() => 1, 'username' => '王五');
                                                echo parent::update($data, $where);
                                                //上面将输出:UPDATE `test` SET `addtime` = ? WHERE `id` = ? AND `username` = ?

                                                //如果我们想要直接执行sql,如下:
                                                $data = array('username' => '张三', 'addtime' => time());
                                                echo parent::update($data, 1)->execute();
                                                //上面将输出:1

    4.获取一条或者多条数据

    通过find($values, $isRows = false, Array $orderBy = array())方法，我们可以获取一条或者多条数据。
    $values 必填：可以是一个主键的值或者主键的值数组，还可以是where条件
    $isRows 选填：返回多行记录还是单行记录，true：多行，false：单行
    $orderBy 选填：当返回多行记录时，可以指定排序。例如：array('time'=>'desc') 或者 array('time'=>'desc','id'=>'asc')
    假如我们子类要调用,如下： 
                                                    $values = 1;
                                                var_dump(parent::find($values));
                                                //上面将输出:array(3) { ["id"]=> string(1) "1" ["username"]=> string(6) "刘一" ["addtime"]=> string(10) "1431356499" }

                                                $values = array(1,2,3);
                                                var_dump(parent::find($values));
                                                /*上面将输出:array(3) { 
                                                        [0]=> array(3) { ["id"]=> string(1) "1" ["username"]=> string(6) "刘一" ["addtime"]=> string(10) "1431356499" } 
                                                        [1]=> array(3) { ["id"]=> string(1) "2" ["username"]=> string(6) "陈二" ["addtime"]=> string(10) "1422806400" } 
                                                        [2]=> array(3) { ["id"]=> string(1) "3" ["username"]=> string(6) "张三" ["addtime"]=> string(10) "1425312000" } 
                                                }*/

                                                $values = array('id' => 1, 'username' => '刘一');
                                                var_dump(parent::find($values));
                                                //上面将输出:array(3) { ["id"]=> string(1) "1" ["username"]=> string(6) "刘一" ["addtime"]=> string(10) "1431356499" }

                                                $values = array('username' => array('刘一', '陈二'));
                                                var_dump(parent::find($values, true));
                                                /*上面将输出:array(2) { 
                                                        [0]=> array(3) { ["id"]=> string(1) "1" ["username"]=> string(6) "刘一" ["addtime"]=> string(10) "1431356499" } 
                                                        [1]=> array(3) { ["id"]=> string(1) "2" ["username"]=> string(6) "陈二" ["addtime"]=> string(10) "1422806400" } 
                                                }*/

                                                $values = array('username' => array('刘一', '陈二'));
                                                var_dump(parent::find($values, true, array('addtime' => 'asc')));
                                                /*上面将输出:array(2) { 
                                                        [0]=> array(3) { ["id"]=> string(1) "2" ["username"]=> string(6) "陈二" ["addtime"]=> string(10) "1422806400" } 
                                                        [1]=> array(3) { ["id"]=> string(1) "1" ["username"]=> string(6) "刘一" ["addtime"]=> string(10) "1431356499" } 
                                                }*/

    5.获取一条或者多条数据

    通过findAll($where = null, Array $orderBy = array(), $limit = null, $fileds = null)方法，我们可以获取一条或者多条数据。
    $where 选填：where条件数组
    $orderBy 选填：排序，比如：array('time'=>'desc')或者array('time'=>'desc','id'=>'asc')
    $limit 选填：limit数量，比如：10
    $fileds 选填：要搜索的字段，比如：id,name。留空默认*
    假如我们子类要调用,如下： 
                                                    $where = array('id' => 1);
                                                var_dump(parent::findAll($where));
                                                //上面将输出:array(1) { [0]=> array(3) { ["id"]=> string(1) "1" ["username"]=> string(6) "刘一" ["addtime"]=> string(10) "1431356499" } }

                                                $where = array('id' => array(1,2,3));
                                                $order = array('addtime' => 'asc');
                                                var_dump(parent::findAll($where, $order));
                                                /*上面将输出:array(3) { 
                                                        [0]=> array(3) { ["id"]=> string(1) "2" ["username"]=> string(6) "陈二" ["addtime"]=> string(10) "1422806400" } 
                                                        [1]=> array(3) { ["id"]=> string(1) "3" ["username"]=> string(6) "张三" ["addtime"]=> string(10) "1425312000" } 
                                                        [2]=> array(3) { ["id"]=> string(1) "1" ["username"]=> string(6) "刘一" ["addtime"]=> string(10) "1431356499" } 
                                                }*/

                                                var_dump(parent::findAll('', array(), 3));
                                                /*上面将输出:array(3) { 
                                                        [0]=> array(3) { ["id"]=> string(1) "1" ["username"]=> string(6) "刘一" ["addtime"]=> string(10) "1431356499" } 
                                                        [1]=> array(3) { ["id"]=> string(1) "2" ["username"]=> string(6) "陈二" ["addtime"]=> string(10) "1422806400" } 
                                                        [2]=> array(3) { ["id"]=> string(1) "3" ["username"]=> string(6) "张三" ["addtime"]=> string(10) "1425312000" } 
                                                }*/

                                                var_dump(parent::findAll('', array(), 1, 'username'));
                                                //上面将输出:array(1) { [0]=> array(1) { ["username"]=> string(6) "刘一" } }

    6.根据条件获取一个字段的值或者数组

    通过findCol($col, $where, $isRows = false, Array $orderBy = array())方法，我们可以根据条件获取一个字段的值或者数组。
    $col 必填：字段名称
    $where 必填：可以是一个主键的值或者主键的值数组，还可以是where条件(这个跟find的$values参数一样)
    $isRows 选填：返回多行记录还是单行记录，true：多行，false：单行
    $orderBy 选填：当返回多行记录时，可以指定排序，比如：array('time'=>'desc')或者array('time'=>'desc','id'=>'asc')
    假如我们子类要调用,如下： 
                                                    $where = 1;
                                                echo parent::findCol('username', $where);
                                                //上面将输出:刘一
                                                
                                                $where = array(1,2,3);
                                                var_dump(parent::findCol('username', $where, true));
                                                //上面将输出:array(3) { [0]=> string(6) "刘一" [1]=> string(6) "陈二" [2]=> string(6) "张三" }

                                                $where = array(1,2,3);
                                                $order = array('addtime' => 'asc');
                                                var_dump(parent::findCol('username', $where, true, $order));
                                                //上面将输出:array(3) { [0]=> string(6) "陈二" [1]=> string(6) "张三" [2]=> string(6) "刘一" }

    7.根据条件删除记录

    通过delete($values, Array $cond = NULL)方法，我们可以根据条件删除记录。
    $values 必填：以是一个主键的值或者主键主键的值数组
    $cond 选填：附加的where条件，关联数组
    假如我们子类要调用,如下： 
                                                    $values = array(1);
                                                echo parent::delete($values);
                                                //上面将输出:1

                                                $values = array(1);
                                                $cond = array('username' => '刘一');
                                                echo parent::delete($values);
                                                //上面将输出:1

    8.分页方法

    通过getPage($page, $pagesize, $url, $fields = '*', Array $where = null, $orderBy = array(), $pageBarOrder = array(1, 2, 3, 4, 5, 6), $pageBarACount = 10)方法，我们可以分页获取数据。
    $page 必填：第几页
    $pagesize 必填：每页多少条
    $url 必填：基础url，里面的{page}会被替换为实际的页码
    $fields 选填：select的字段，全部用*，多个字段用逗号分隔
    $where 选填：where条件，关联数组
    $orderBy 选填：排序字段，比如：array('time'=>'desc')或者array('time'=>'desc','id'=>'asc')
    $pageBarOrder 选填：分页条组成，可以参考系统方法手册中分页条方法
    $pageBarACount 选填：分页条a的数量，可以参考系统方法手册中分页条方法 
                                                    var_dump(parent::getPage(1, 2, '#'));

                                                        ["page"]=> "分页代码" 
                                                }*/

                                                /*上面将输出:array(2) { 
                                                        ["items"]=> array(3) { 
                                                                [0]=> array(1) { ["username"]=> string(6) "刘一" }

  4. 使用Dao层

    可以在任意地方使用下面的代码加载一个dao层类：
    示例如下： 
                                                    Sr::dao('TestDao');
    我们可以看到上面有一个参数，这个是什么意思呢，是加载dao类的时候，不需要前缀Dao_，
    比如上面的：Dao_TestDao，加载的时候只用Sr::dao('TestDao');
    例如:我们要创建Dao类文件名为ListDao.php，类名就是Dao_ListDao，那么参数就要输入ListDao

  5. 示例一

    比如有文件：classes/Dao/ArticleUser.php
    那么ArticleUser.php文件里面的类名就应该是：Dao_ArticleUser。
    那么要加载Dao类如下: 
                                                    Sr::dao('ArticleUser');

  6. 示例二

    比如有文件：classes/Dao/Vip/User.php
    那么User.php文件里面的类名就应该是：Dao_Vip_User,也就是下划线代表着文件夹的分隔符。
    那么要加载Dao类如下: 
                                                    Sr::dao('Vip_User');
                                                //还有另一种方式例如以下
                                                Sr::dao('Vip/User.php');
                                                //我们也可以不用带.php后缀,例如以下：
                                                Sr::dao('Vip/User');

    //将输出：刘一
    为什么会输出“刘一”?
    我们可以从上面例子看Controller_Welcome控制器类，
    在do_dao方法里面做了加载Dao层中的Dao_User类赋值给$dao。
    然后通过$dao输出Dao_User类中的getUserName方法,
    getUserName方法，主要是去数据库查询users表查询username字段的第一条数据，最后得到查询结果数据数组中的username字段。
    详细请看数据库手册的“查询数据”和“查询结果集的使用”，
    所以浏览器才会显示“刘一”。
